NOTES and issues: 
-You need to have the Table.tbl in your working dir!
-You need to have a NEW version of ndstool in your $path
-You have to remove the change="false" attributes or change them to true in the xml files
you actually want to change
-You can actually write xml files from scratch if you know the IDs
-Type nothing or help in the tool for help 
-PROTIP: this tool does not use normal commandline the reason for this is quite simple:
to avoid a massiv ammonts of "the tool closes immedeately" complaints console apps
usually get. You can still script it like this: thenewpoketext < commands.txt
see batch.cmd and script.txt
